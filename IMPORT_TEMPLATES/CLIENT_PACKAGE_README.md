## 🚀 **How Client Should Use This**

### **Step 1: Download Templates**
- Download all 5 CSV files
- Open them in Excel
- Save as Excel files (.xlsx)

### **Step 2: Fill in Order**
1. **SUPPLIERS** (List all supplier companies)
2. **RAW_MATERIALS** (List all materials, match supplier names exactly)
3. **PRODUCTS** (List all finished products)
4. **PRODUCT_RECIPES** (Define what materials make each product)
5. **INDIVIDUAL_PRODUCTS** (Specify quantities to create)

### **Step 3: Check Names Match**
- Supplier names must match exactly between sheets 1 and 2
- Product names must match exactly between sheets 3, 4, and 5
- Material names must match exactly between sheets 2 and 4

### **Step 4: Submit to You**
- Send back the filled Excel files
- You'll import them into the system

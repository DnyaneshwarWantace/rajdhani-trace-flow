# 🚀 **First Time Inventory Setup Guide**

## 📋 **What You Need to Import First**

To set up your inventory for the first time, you need to import these files **IN THIS ORDER**:

### **Step 1: Import Suppliers** 📞
**File:** `ACTUAL_SUPPLIERS_TEMPLATE.csv`
- **What it contains:** 8 real suppliers with complete contact details
- **Why first:** Raw materials need supplier information
- **How to import:** Use the "Import Suppliers" feature in the system

### **Step 2: Import Raw Materials** 🧵
**File:** `ACTUAL_RAW_MATERIALS_TEMPLATE.csv`
- **What it contains:** 15 real raw materials with actual stock levels
- **Why second:** Products need materials for recipes
- **How to import:** Use the "Import Raw Materials" feature in the system

### **Step 3: Import Products** 🏺
**File:** `ACTUAL_PRODUCTS_TEMPLATE.csv`
- **What it contains:** 10 real products with dimensions and pricing
- **Why third:** Need products before creating recipes
- **How to import:** Use the "Import Products" feature in the system

### **Step 4: Import Product Recipes** 🍳
**File:** `ACTUAL_PRODUCT_RECIPES_TEMPLATE.csv`
- **What it contains:** 24 recipe entries linking products to materials
- **Why last:** Need both products and materials to create recipes
- **How to import:** Use the "Import Product Recipes" feature in the system

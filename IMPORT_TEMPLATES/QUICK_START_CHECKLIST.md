## 🚨 **Common Issues & Solutions:**

### **Import Errors:**
- [ ] **Missing suppliers:** Import suppliers first
- [ ] **Invalid data:** Check CSV format and required fields
- [ ] **Duplicate entries:** Clear existing data before import
- [ ] **File format:** Ensure CSV files are properly formatted

### **Data Issues:**
- [ ] **Wrong quantities:** Update stock levels manually
- [ ] **Missing suppliers:** Add missing suppliers first
- [ ] **Invalid recipes:** Check material names match exactly
- [ ] **Pricing errors:** Update cost per unit manually
